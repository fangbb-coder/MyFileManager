# Git安装指南

在将项目发布到GitHub之前，需要先在您的系统上安装Git。请按照以下步骤操作：

## Windows系统安装Git

1. 访问官方Git下载页面：https://git-scm.com/download/win
2. 下载适合您系统的最新版本Git安装程序
3. 运行安装程序，按照默认设置完成安装
4. 安装完成后，重新打开命令提示符或PowerShell
5. 运行以下命令验证安装：
   ```
   git --version
   ```

## 安装完成后继续操作

安装Git后，我们将继续以下步骤：

1. 配置Git用户名和邮箱
2. 将项目文件添加到Git仓库
3. 提交初始版本
4. 创建GitHub仓库并推送到远程服务器

请在安装完成后告知我，我们可以继续后续操作。
